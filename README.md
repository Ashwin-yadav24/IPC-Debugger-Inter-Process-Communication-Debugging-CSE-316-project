# IPC Debugger – Inter Process Communication Debugging Tool

## 📌 Overview

The **IPC Debugger** is a Python-based tool designed to simulate and visualize different **Inter Process Communication (IPC)** mechanisms. It helps students understand how processes communicate and how issues like deadlocks occur in operating systems.

---

## 🚀 Features

* 🔄 Pipe Communication Simulation
* 🧵 Message Queue Simulation
* 🧠 Shared Memory Simulation
* ⚠️ Deadlock Visualization
* 🖥️ GUI-based Interface for better understanding

---

## 🛠️ Technologies Used

* Python
* Tkinter (GUI)
* OS Concepts (IPC mechanisms)

---

## 📂 Project Structure

```
IPC-Debugger/
```
